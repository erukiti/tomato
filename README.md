# Tomato

React + Redux の習作で作ってるポモドーロタイマーです。

* Electron
* ES2015
* React
* Redux
* VSCode ready

# 動かし方

```
$ npm i
$ npm start
```

他にも

```
$ npm run release
```

や

```
$ npm test
```

などもございます。
